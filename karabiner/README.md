# How to use

1. ln -snf ~/dotfiles/karabiner/rules  ~/.config/karabiner/assets/complex_modifications
2. Open karabiner GUI, Complex Modifications → Add predefined rule → Find HyperGlyph and click `Enable All`.

# How to update
1. Delete all of the previous rules in Complex Modifications.
2. Repeat the step 2 in the "How to use" section.
